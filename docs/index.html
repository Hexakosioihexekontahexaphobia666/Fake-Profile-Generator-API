Fake Data API

Generate realistic fake profiles for testing and security purposes.

Authentication Endpoints Code Examples FAQ

🔑 Authentication

To use this API, you must obtain an API key. Pass your key as a header:

Headers: {"X-API-Key": "your-api-key-here"}

📌 API Endpoints

1️⃣ Generate Fake Profile

Creates a fake user profile.

GET /generate?age=25&gender=male&country=US

2️⃣ Bulk Generate Profiles

Generates multiple fake profiles (max 100).

GET /bulk-generate?count=5

3️⃣ Get Supported Countries

GET /countries

4️⃣ View API Usage Stats

GET /stats

🚀 Code Examples

📌 Python


import requests
headers = {"X-API-Key": "your-api-key-here"}
response = requests.get("http://127.0.0.1:8000/generate", headers=headers)
print(response.json())

📌 JavaScript (Fetch)


fetch("http://127.0.0.1:8000/generate", {
    headers: { "X-API-Key": "your-api-key-here" }
}).then(res => res.json()).then(data => console.log(data));

📌 PHP


<?php
$headers = array("X-API-Key: your-api-key-here");
$ch = curl_init("http://127.0.0.1:8000/generate");
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
echo $response;
?>

📌 cURL


curl -H "X-API-Key: your-api-key-here" http://127.0.0.1:8000/generate

❓ Frequently Asked Questions

What is this API for?

This API generates realistic fake data for testing and security purposes.

How do I get an API key?

Register and generate an API key using `/register` and `/generate-api-key`.

Is there a rate limit?

Yes, the free tier allows 5 requests per minute.
